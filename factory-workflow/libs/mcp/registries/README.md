# MCP Registries

## O que sao
Registries sao catalogos de componentes reutilizaveis.

## Registry vs design-system
- Registry: fonte externa ou catalogo de componentes.
- Design-system: fonte interna de tokens, componentes e padroes.

## Quando usar registry
- Sempre que houver componente equivalente.
- Antes de criar novo componente.

## Ordem obrigatoria de busca
1. Registries aprovados.
2. Design-system interno.
3. Criacao nova (ultimo recurso).

## Evidencias em PR
- Registro da busca no registry.
- Justificativa para criacao nova.
