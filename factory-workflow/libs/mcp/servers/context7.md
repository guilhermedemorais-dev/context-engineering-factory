# Context7 (MCP Server)

## Descricao
Context7 e um servidor MCP para documentacao confiavel, APIs e referencias tecnicas.

## Problemas que resolve
- Acesso a documentacao atualizada.
- Verificacao de APIs e comportamentos.
- Evitar suposicoes.

## Quando o uso e obrigatorio
- Sempre que uma implementacao depender de API/documentacao externa.
- Sempre que houver risco de alucinacao tecnica.

## Quando NAO usar
- Quando o contexto interno ja define tudo.
- Quando a informacao nao e relevante para a tarefa.

## Exemplos de tarefas
- Validar como uma API externa deve ser chamada.
- Confirmar comportamento de uma biblioteca.
- Consultar padroes recomendados.

## Limitacoes e riscos
- Informacao pode estar incompleta se a fonte estiver desatualizada.
- Requer validacao com o contexto interno.

## Evidencia em PR
- Link/citacao do trecho consultado.
- Resumo do impacto na implementacao.
