# Color

## Paleta base
- Neutros em niveis (ex.: 50..900).
- Feedback: success, warn, error, info em niveis.

## Tokens semanticos (conceitual)
- bg, fg, border
- primary, secondary
- muted, subtle
- destructive

## Regras de uso
- Contraste adequado entre texto e fundo (ver `factory-workflow/design-system/a11y.md`).
- Nunca usar cor como unico indicador de estado.
- Usar tokens semanticos para consistencia.

## Extensao da paleta
- Adicionar novos tokens mantendo escala e nomenclatura.
- Evitar duplicar significado.
- Registrar mudancas no changelog do arquivo.
