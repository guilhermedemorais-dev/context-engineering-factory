# Integration Tests

## O que e integracao
Contratos entre modulos, servicos e bases de dados.

## Estrategia de ambientes
- Ambientes isolados.
- Dependencias simuladas quando possivel.
- Dados controlados.

## Evidencias e criterios
- Relatorios de integracao.
- Logs de falha com contexto.
- Pass/fail registrado no PR.
