# Design System Example - Agendamento Simples

## Tokens
- Cor: neutros + feedback (success/warn/error).
- Spacing: escala discreta.
- Typography: titulos, corpo e legendas.

## Componentes base
- Buttons, inputs, select, modal.
- Estados: default/hover/focus/disabled/loading/error.

## Acessibilidade
- Contraste adequado.
- Navegacao por teclado.

## Politica
- Registry-first e aderencia a tokens.
