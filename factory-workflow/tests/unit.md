# Unit Tests

## O que testar
- Regras puras.
- Validacoes deterministicas.
- Transformacoes de dados.

## O que nao e unit
- Integracoes externas.
- Dependencias de rede.
- Fluxos ponta a ponta.

## Criterios e evidencias
- Relatorio de testes unitarios.
- Cobertura quando aplicavel.

## Como reportar falhas
- Descrever caso e impacto.
- Apontar modulo afetado.
- Registrar evidencias.
