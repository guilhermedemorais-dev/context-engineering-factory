# shadcn-vue (Registry)

## Descricao
Registry aprovado para componentes UI reutilizaveis.

## Tipo de componentes
Componentes de interface genericos e padronizados.

## Quando usar
- Quando houver componente equivalente ao necessario.
- Quando precisar de base consistente para UI.

## Quando NAO usar
- Quando o componente exigir comportamento especifico nao suportado.
- Quando houver conflito com o design-system.

## Regras de adaptacao
- Aplicar tokens do `factory-workflow/design-system/*`.
- Manter acessibilidade minima.

## Evidencia minima em PR
- Link para o componente utilizado.
- Nota de adaptacoes realizadas.
