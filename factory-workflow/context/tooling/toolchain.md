# Toolchain

## Conceito
Conjunto de ferramentas usadas no ciclo de desenvolvimento, agnostico de stack.

## Categorias
- desenvolvimento
- testes
- qualidade
- seguranca
- CI/CD
- observabilidade

## Regras
- Ferramentas sao exemplos, nao obrigatorias.
- Decisoes devem respeitar principios e guardrails do contexto.

## Criterios para nova ferramenta
- Resolver problema claro e documentado.
- Compatibilidade com contexto e quality bars.
- Impacto de manutencao aceitavel.
- Decisao registrada quando necessario.

## Relacao com CI/CD
- Toolchain deve mapear para jobs em `factory-workflow/cicd/templates.md`.
