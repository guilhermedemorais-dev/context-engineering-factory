# Tokens

## O que sao
Tokens sao valores base que garantem consistencia visual e funcional. Componentes usam tokens; tokens sao a fonte de verdade.

## Taxonomia
- Core tokens: valores brutos (ex.: escala de espacamento, tamanhos base).
- Semantic tokens: aplicacoes com significado (ex.: bg-primary, fg-muted).

## Regras
- Nomes consistentes e descritivos.
- Unidades coerentes dentro da mesma categoria.
- Escalas regulares (incrementos previsiveis).

## Politica
- Componentes nao devem usar valores diretos; devem referenciar tokens.
- Mudanca em token requer revisao de impacto.
