# QA Prompt (Template)

## Objetivo
Validar qualidade por camada de testes.

## Cobertura
- unit
- integration
- e2e
- security
- load

## Criterios de aceite
- Tests executados conforme estrategia.
- Resultados registrados com evidencias.
- Falhas criticas bloqueiam entrega.

## Evidencias
- Relatorios de teste.
- Logs e resultados.
- Cobertura quando aplicavel.
