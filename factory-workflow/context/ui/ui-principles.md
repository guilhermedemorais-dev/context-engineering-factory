# UI Principles

## Principios
- Clareza > estetica.
- Consistencia > novidade.
- Acessibilidade > customizacao.

## Prioridades em conflito
- Acessibilidade vence sempre.
- Consistencia vence preferencias locais.
- Clareza vence ornamentacao.

## Anti-padroes
- Componentes inconsistentes sem justificativa.
- Contraste insuficiente.
- Navegacao apenas por mouse.
- Interface que exige memorizacao do usuario.

## Relacao com design-system
- Tokens e componentes de `factory-workflow/design-system/*` sao a base.
- Customizacao so quando necessario e documentado.
