# StackOverflow (MCP Server)

## Proposito
Consulta pratica para casos de borda, erros especificos e comportamento observado pela comunidade.

## Quando usar
- Erros incomuns sem cobertura clara em docs oficiais.
- Edge cases de linguagem, framework ou tooling.
- Confirmacao rapida de comportamento observado.

## Quando nao usar
- Como unica fonte para decisoes criticas.
- Para substituir documentacao oficial.

## Criterios de confianca
- Preferir respostas com score alto.
- Preferir respostas recentes quando houver mudanca de versao.
- Sempre cruzar com docs oficiais (ex.: Context7).

## Evidencias minimas
- Registrar link da pergunta e da resposta utilizada.
- Registrar trecho relevante (sem copiar codigo inteiro).
- Consolidar no `research.md`/`plan.md`.
